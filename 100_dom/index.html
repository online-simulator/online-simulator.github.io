DOM Document Object Model
・プログラムから文書を操作するインターフェースを提供する
・文書をオブジェクトのツリー構造で表すモデル体系
・協調して動作する複数のAPIから構成される
　文書　⇔　複数のAPI　⇔　JavaScript
EventTarget Nodeの親となるインターフェース
window等
Node Elementの親となるインターフェース
プロパティ: textContent等
Element HTMLElementの親となるインターフェース
プロパティ: innerHTML等
HTMLElement 任意のHTML要素を表すインターフェース
プロパティ: innerText他、親から継承した全プロパティ
要素ノード
テキストノード 		文書内の任意タグで記述される要素のオブジェクト
